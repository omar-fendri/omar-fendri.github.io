# omarfendri.com

Personal portfolio for Omar Fendri. The site is intentionally dependency-free: semantic HTML, modern CSS, and a small amount of JavaScript.

## Local development

```bash
npm run serve
```

Open [http://localhost:4173](http://localhost:4173).

## Checks

```bash
npm run check
```

The site deploys from `site/` through the GitHub Pages workflow on every push to `main`.
